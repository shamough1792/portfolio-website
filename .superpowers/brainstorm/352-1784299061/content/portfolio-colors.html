🎨 顏色隨你換 — CSS Variables

整站顏色定義在 root，改三行就換一套主題

💡 核心概念

CSS 變數定義在 :root，換色不用到處改，三行搞定 

:root {
  --bg-primary: #1a1a2e;      /* 主背景 */
  --bg-secondary: #16213e;    /* 卡片背景 */
  --bg-accent: #0f3460;       /* 強調區塊 */
  --accent: #e94560;          /* 主強調色 */
  --text-primary: #ffffff;    /* 主文字 */
  --text-secondary: #a0a0b8;  /* 次要文字 */
  --text-muted: #555555;      /* 弱化文字 */
}
